;===========================================================================
;=                      Entry Syntax                                       =
;===========================================================================
;
; Begin
;	TargetFile	filename without path the entry gets written to
;	SourceFile	filename without path the entry has as source data
;	TargetPath	the path of the target (can use variables, see below)
;	Disk		the number of the disk this entry or the first segment of this entry is located on, default = 1
;	LeaveOnMinimum	when this is 1, the file does not get installed when the use choose a minumum install, default = 0
;	SkipIfNotFound  when this is 1, the file gets skipped if it cannot be found. Default behavior will show an error when files cannot be found. default = 0
;	NumSegments	when this is >0, the entry consists of multiple segments of the same name on different disks, default = 0
;	InstallType	the type of entry:
;				copy		the file gets copied from source to target
;				decompress	the file gets decompressed from source to target
;	
; End
;
; Variables:
;
;	%install_path%	the path the user choose to install the game to
;
